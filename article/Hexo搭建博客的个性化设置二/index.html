为项目主页添加 README

在 Github 上的博客仓库主页空荡荡的，没有 README。如果把 README.md 放入 source 文件夹，hexo g 生成时会被解析成 html 文件，放到public文件夹，生成时又会自动删除。

解决方法很简单，在站点配置文件中，搜索 skip_render:，在其冒号后加一个空格然后加上 README.md 即可。
